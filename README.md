# Access Link

ACCESS 장비 이미지와 매뉴얼 자료를 정리하기 위한 작업 공간입니다.

## 현재 구성

- `ACCESS 장비 이미지 및 메뉴얼/`
  - KakaoTalk으로 받은 ACCESS 장비 이미지 파일

## 작업 순서

1. 원본 이미지는 그대로 보관합니다.
2. 이미지별 장비명, 메뉴 위치, 설정값을 확인합니다.
3. 확인한 내용을 문서로 정리합니다.
4. 필요한 경우 GitHub 저장소에 업로드해 변경 이력을 관리합니다.

## 주의사항

- 원본 파일명은 임의로 변경하지 않습니다.
- 삭제나 덮어쓰기가 필요한 작업은 먼저 확인합니다.
- 오류가 발생하면 원인, 발생 위치, 해결 방법을 기록합니다.
