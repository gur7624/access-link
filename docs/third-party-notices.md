# 서드파티 고지

## FaceNet TFLite 모델

- 출처: https://github.com/shubham0204/FaceRecognition_With_FaceNet_Android
- 파일: `AccessLinkTester/app/src/main/assets/facenet.tflite`
- 라이선스: Apache License 2.0
- 용도: 전면 카메라에서 검출한 얼굴 이미지를 128차원 임베딩으로 변환해 등록 얼굴과 비교한다.

해당 예제의 README는 FaceNet 기반 판별 정확도에 한계가 있을 수 있음을 명시한다.
운영 환경에서는 현장 조명, 카메라 위치, 등록 이미지 품질 기준을 별도로 검증해야 한다.
